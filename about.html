Hi, I'm Mohan.

B.Tech Computer Science student building agentic AI systems that actually ship — not demos that only work in a notebook.

I'm currently a B.Tech Computer Science and Engineering student at GLA University, Mathura (Class of 2027), building toward a career in software engineering with a focus on agentic AI systems. My core stack is MERN in plain JavaScript, layered with LangGraph.js, LangChain.js, and the Gemini SDK — I like building multi-agent pipelines that do real work: reviewing pull requests, tutoring through competitive programming problems, or in this case, giving a coding agent a memory that outlives a single terminal session. Repo Brain was built for the WeMakeDevs × Cognee hackathon, where the brief was simple — build anything meaningful on top of Cognee's memory lifecycle — and the gap I kept noticing on real teams was that every developer's agent starts every session from zero, even when a teammate's agent solved the exact same problem an hour earlier.
